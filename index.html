HOW TO PLAY

HOLD Touch / Click / Space

Tether to nearest black hole and orbit (slow-mo active)

RELEASE Let go / Release Space

Slingshot forward in current direction

STEER Arrows / Swipe

Left/Right steer, Up/Down thrust/brake after releasing

ASSIST FEATURES

  • 🛡️ Shield - Saves you once per run
  • 🎯 Trajectory - Shows predicted path
  • 🧲 Gravity Assist - Gentle pull to wells
  • ⏳ Slow-Mo - Time slows while holding

TIPS

  • Watch the trajectory preview dots
  • Orbit longer to build up speed
  • Use slow-mo to time your release
  • Shield shows as 🛡️ in the indicator
